# Cursor Boilerplate Agent instructions

> Guidance for autonomous coding agents (OpenAI Codex CLI, Copilot Agent Mode, Cursor, etc.)
> Read this before writing, editing, or executing anything in this repo.

This is a frontend boilerplate for designers to start working with cursor and create UI prototypes for user testing, proofs of concept, and rapid iteration, using our react component library.

## Dev instructions
- When working in this frontend boilerplate always use components within our design system, UXCore @ux/, as much as possible.
- Always refer to `uxcore` MCP for context on component documentation.
- For design guidance, i.e. when to use which components and how exactly they should be applied, always refer to `design-guidance.md`. Use it to respond to any questions related to the design system

## Code style guidelines
- Prioritize design system usage, accessibility, and usability
- Ensure any attached UI is matched with high fidelity, but without
